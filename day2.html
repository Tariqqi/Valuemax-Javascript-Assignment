Objects & Arrays in JavaScript

Objects are collections of related data and functions, stored as key-value pairs. Keys are also called properties or attributes, and values can be any data type, including other objects.

Creating an Object

We can create an object using object literal syntax.

// Using object literal syntax
const personTaiwo = {
  firstName: "Taiwo",
  lastName: "Ojo",
  age: 25,
  greet: function() {
    console.log("Hello, my name is " + this.firstName + " " + this.lastName);
  }
};

const personEmmanuel = {
  firstName: "Emmanuel",
  lastName: "Ibrahim",
  age: 30,
  greet: function() {
    console.log("Hello, my name is " + this.firstName + " " + this.lastName);
  }
};

Accessing Object Properties

You can access object properties using dot notation or bracket notation. 

// Dot notation
console.log(personTaiwo.firstName); // Taiwo
personTaiwo.greet(); // Hello, my name is Taiwo Ojo

// Bracket notation
console.log(personEmmanuel['lastName']); // Ibrahim
personEmmanuel['greet'](); // Hello, my name is Emmanuel Ibrahim

Modifying Object Properties

We can add, update, or delete properties of an object.

// Add a new property
personTaiwo.hobby = "Reading";
console.log(personTaiwo.hobby); // Reading

// Update an existing property
personEmmanuel.age = 31;
console.log(personEmmanuel.age); // 31

// Delete a property
delete personTaiwo.age;
console.log(personTaiwo.age); // undefined

Iterating over Object Properties

We can iterate over the properties of an object using a for...in loop. 

for (let key in personTaiwo) {
  console.log(key + ": " + personTaiwo[key]);
}

Arrays in JavaScript

Arrays are ordered collections of values, where each value is called an element and has an index.

Creating an Array

We can create arrays using array literal syntax.

const names = ["Taiwo", "Emmanuel", "Kemi"];
const ages = [25, 30, 27];

Accessing Array Elements

We can access array elements using their index.

console.log(names[0]); // Taiwo
console.log(ages[1]); // 30

Modifying Array Elements

We can add, update, or remove elements from an array.

// Add elements
names.push("Tunde");
console.log(names); // ["Taiwo", "Emmanuel", "Kemi", "Tunde"]

// Update elements
ages[0] = 26;
console.log(ages); // [26, 30, 27]

// Remove elements
names.pop();
console.log(names); // ["Taiwo", "Emmanuel", "Kemi"]

Iterating over Array Elements

We can iterate over array elements using a for loop or for...of loop. 

for (let i = 0; i < names.length; i++) {
  console.log(names[i]);
}

for (let name of names) {
  console.log(name);
}

Common Array Methods

  • push(): Adds one or more elements to the end of an array.
  • pop(): Removes the last element from an array.
  • shift(): Removes the first element from an array.
  • unshift(): Adds one or more elements to the beginning of an array.
  • slice(): Returns a shallow copy of a portion of an array.
  • splice(): Adds or removes elements from an array.

Conclusion

Understanding objects and arrays is fundamental to mastering JavaScript. They are used to store and manipulate data, making our code more dynamic and powerful.
